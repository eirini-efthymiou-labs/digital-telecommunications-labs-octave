# Lab 1 - Introduction to GNU Octave

This folder contains the first laboratory exercise for the course **Introduction to Digital Telecommunications**.

## Topics covered

- Basic GNU Octave commands
- User input with `input`
- Basic calculations
- Working with vectors
- Formatted output with `fprintf`
- Simple telecommunications-related formulas

## Goal

The goal of this lab is to practice basic Octave programming and to organize the solution in a clear and readable way.

## Files

- `LAB_1_504_TELECOM.m`: Main Octave script for Lab 1
